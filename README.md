# books_bot
Библиотека телеграмм бот

# Установка
**Версия python не меньше 3.10**

  1. Установка библиотек
    ```python3 -m pip install -r requirements.txt```
  2. Настройка бота
     ```В файле config.py в переменную bot_token = '' вставить свой токен (@BotFather)```
  3. Запуск
     ```python3 main.py```
